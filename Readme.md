# Python CLI: JSON CRUD


### Setting up the project folder
> python -m venv venv

### Activating the environment
> source venv/Scripts/activate

### Installing the dependencies
> pip install -r requirements.txt

### Running the application
> python cli.py

---
- [x] Listing users:
